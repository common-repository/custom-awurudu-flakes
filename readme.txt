== Short Description ==
Adds a delightful falling Avurudu Flake effect representing traditional Sinhala Avurudu food items to your WordPress site.

== Description ==
**Custom avurudu flakes** adds a delightful falling avurudu flakes effect to your WordPress site, celebrating the Sinhala avurudu festival. Each flake represents a traditional Sri Lankan food item, including:

- Kavum
- Kokis
- Asmi
- Aluwa
- Mung Kavum
- Seeni Murukku
- Banana

Just activate the plugin to see the festive flakes on every page!

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/avurudu` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy the falling avurudu flakes effect celebrating Sinhala avurudu!

== Frequently Asked Questions ==
= Do I need to do anything after activating the plugin? =
Nope! Simply activating the plugin will start the avurudu flakes effect on your site.

= Will there be options to customize the avurudu flakes in the future? =
Yes! Future updates will provide options to control the number, speed, and other effects of the avurudu flakes.

= Are the avurudu flake images included in the plugin? =
Yes, all the traditional food item images are included within the plugin files.
